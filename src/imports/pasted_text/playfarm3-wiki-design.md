플레이팜3 공식 위키 웹사이트 메인 페이지를 디자인해줘.
전체 분위기는 밝고 깔끔한 버전으로, 게임 위키 + 공식 안내 사이트 + 커뮤니티형 정보 페이지 느낌으로 제작해줘.
디자인 스타일은 all-d-ding.vercel.app와 유사한 구조를 참고하되, 더 밝고 부드러운 색감, 게임 서버 위키 느낌, 한국어 중심 UI로 구성해줘.

전체 컨셉

웹사이트 유형: 플레이팜3 공식 위키 / 가이드 사이트

분위기: 밝음, 친근함, 깔끔함, 보기 쉬움, 초보자 친화적

핵심 키워드: 게임 위키, 생활형 콘텐츠, 힐링, 커뮤니티, 정보 정리, 초보자 친화

레이아웃: 상단 고정 헤더 + 메인 히어로 섹션 + 카테고리별 정보 카드 + 위키형 문서 섹션

스타일: 라운드 카드 UI, 은은한 그림자, 넓은 여백, 부드러운 블루/민트/화이트 계열

톤앤매너: 마인크래프트 생활 서버 느낌 + 정리 잘 된 공식 위키

컬러 방향

메인 배경: 화이트 또는 아주 연한 블루

포인트 컬러: 스카이블루, 민트, 연한 그린, 소프트 옐로우

카드 배경: 화이트

텍스트: 진한 회색

전체적으로 채도가 너무 강하지 않게, 눈이 편한 색감 사용

다크모드 말고 밝은 버전으로 제작

폰트 / 분위기

한글 가독성이 좋은 현대적인 웹 폰트 느낌

제목은 굵고 명확하게

본문은 편안하게 읽히는 크기

너무 딱딱한 위키보다 공식 안내 사이트처럼 친절한 느낌
